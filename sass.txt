=> Interpolation : 

    scss:   
    .main{
        display:block;
        #{&}__inner {                       where : & represents the direct parent
            width:100%;                           : #{&} called interpolation
        }
    }

    css:    
    .main {
        display:block
    }
    .main .main__inner{
        width:100%;
    }


=> Variable with key pair value:

    $font-weights : (
        "regular" : 400,
        "medium" : 500,
        "semibold" : 600,
        "bold" : 700,
    )

    use: 

    .main{
        font-weight: map-get($font-weights, regular);
    }

    css:
    .main{
        font-weight: 400;
    }


=> Functions : Use to compute values and return values

   @function weight($weight-name) {
       @return map-get($font-weights, $weight-name);
   }

   use:

   .main{
       font-weight : weight(bold);
   }

   css:
   .main{
        font-weight: 400;
    }


=> mixins: Use to define styles

    @mixin flexBox {
        display: flex;
        flex-wrap: wrap;
    }

    use:
    .main{
        @include flexBox;
    }

    css:
    .main{
        display: flex;
        flex-wrap: wrap;        
    }

    -> With passing arguments:

        @mixin flexBox($direction) {
            display: flex;
            flex-wrap: wrap;
            flex-direction : $direction;
        }

        use:
        .main{
            @include flexBox(row);
        }

        css:
        .main{
            display: flex;
            flex-wrap: wrap;        
            flex-direction : row;
        }


=> extend: Use to add the same styling elsewhere    

    1)
    .main{
        display:block;
        #{&}__inner {
            width:100%;
        }
        #{&}__inner1 {
            @extend .main__inner;
            display : block;
        }
    }

    css:
    .main .main__inner, .main .main__inner1{
        width:100%;
    }
    .main .main__inner1{
        display : block;
    }

    2)
    %toolbelt {
        box-sizing: border-box;
        width: 100%;
    }
    .action-buttons {
        @extend %toolbelt;
    }

    css:
    .action-buttons {
        box-sizing: border-box;
        width: 100%;
    }


=> @at-root: When there is a multiple nesting targets and if we want to emit parent target then this rule is used.

    scss:
    ul {
        list-style: none;
        li {
            position: relative;
            @at-root a {
                display: block;
            }  
        }
    }

    css:
    ul{
        list-style: none;
    }
    ul li{
        position: relative;
    }
    a{
        display:block
    }
